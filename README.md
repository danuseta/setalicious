## Prasyarat
- Node.js (v16+)
- npm

## Instalasi
1. Clone repositori
2. Pasang dependensi:
```bash
npm install
```

3. Skrip pengembangan:
```bash
# Jalankan server pengembangan
npm run start-dev

# Bangun untuk produksi
npm run build

# Layani versi produksi
npm run serve
```
